# Practice Java

This repository contains all exercises and practice code implemented throughout a Java course.  
It covers foundational topics and practical examples to demonstrate understanding of core Java concepts.

---

## Description

The project is organized by topics that mirror the course blocks, with each exercise implemented in Java.  
It demonstrates practical use of Java fundamentals, object-oriented programming, and exception handling.

---

## Topics Covered

- Java Introduction: variables and data types
- Operators
- Strings
- Conditional statements
- Data structures
- Loops
- Methods and functions
- Object-Oriented Programming
- Exception handling
- Extras and additional exercises

---

## Technologies

- Java SE — programming language  
- Gradle — build and dependency management
